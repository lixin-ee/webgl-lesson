# webgl-lesson
webgl lesson work
readMe-zh：

计算机图形学作业，内容为webgl编程.

hw1：

内容:画一只平面上的动物,完成鼠标拖拽(整体或部分)，键盘控制位置，菜单控制位置和旋转角度。

完成情况:

使用webgl-utils,webgl-lessons画一只狗以完成此任务。

基于https://webglfundamentals.org/webgl/lessons/zh_cn/上提供的内容编程
